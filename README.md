2D Graphics Editor in C
A basic console-based graphics editor that uses a 2D character matrix to create and manipulate shapes.
Features
Draw Rectangle
Draw Line
Draw Triangle
Draw Circle
Remove Shapes
View Canvas
Technologies
C Language
Arrays
Structures
Functions
How to Run
```bash
gcc graphic_editor.c -o graphic_editor
./graphic_editor
```
